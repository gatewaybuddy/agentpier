# CORS Wildcard Fix — 2026-02-15

## What Changed
- **`src/utils/response.py`**: Replaced `Access-Control-Allow-Origin: *` with `https://agentpier.org` across all response helpers (`success`, `error`, `too_many_requests`). Extracted into `ALLOWED_ORIGIN` constant and `_cors_headers()` helper for single point of change.
- **`infra/template.yaml`**: Updated API Gateway CORS `AllowOrigin` from `'*'` to `'https://agentpier.org'`.

## Why
Critical security finding C1 from `docs/security/2026-02-15-audit.md`. Wildcard CORS allows any website to make authenticated cross-origin requests to the API.
